## Python FastAPI

## Run docker compose
1. Go to the API folder and run the following command:
```bash
docker compose up -d
```
2. Wait to build images and run the API.
3. API will be available on [http://localhost:8000/](http://localhost:8000/)
4. Swagger documentation available on  [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)
